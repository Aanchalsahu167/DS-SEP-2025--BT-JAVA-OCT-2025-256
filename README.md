A robust and user-friendly **Trip Management System** built using Java Spring Boot and Thymeleaf. This project allows users to plan trips, track expenses, and monitor trip statuses through an interactive dashboard.

* **Full CRUD Functionality**: Add, View, Update, and Delete trip records.

* **Real-time Search**: Instant search functionality to filter trips by Name or Destination.
* **Dynamic Budget Counter**: Automatically calculates the total estimated budget for all planned trips.
* **Responsive UI**: Modern and clean interface built with Bootstrap 5.
* **H2 Database Integration**: Uses an in-memory database for fast development and testing.
